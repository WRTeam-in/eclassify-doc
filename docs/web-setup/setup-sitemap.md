# How to Setup Sitemap

1. **Configure Web URL**: Add your web domain or subdomain URL in the `.env` file. This is essential for generating correct links within the sitemap.

   ![Web URL](/images/web/web_url.png)

2. **Automatic Generation**: The sitemap is automatically generated at **build time**. Additionally, it is configured to be automatically updated **weekly** to ensure that new content and listings are reflected.
